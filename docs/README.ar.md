# خريطة أرقام آيات القراءات

**مجموعة بيانات مفتوحة المصدر لربط أرقام الآيات بين أنظمة العدّ الستة المعتمدة في القراءات العشر.**

[Read this document in English](../README.md)

---

## خلفية

نُقل القرآن الكريم عبر **10 قراءات** متواترة، ولكل قراءة **راويان** — فيكون المجموع 20 رواية. فمثلاً: قراءة عاصم رواها حفص وشعبة.

تتبع كل قراءة أحد **أنظمة العدّ الستة** التي تحدّد مواضع فواصل الآيات. وقد تشترك عدة قراءات في نظام عدّ واحد:

- **الكوفي** — عاصم (حفص وشعبة)، حمزة، الكسائي، خلف
- **المدني الأخير** — نافع (ورش وقالون)
- **البصري** — أبو عمرو، يعقوب
- **المكي** — ابن كثير
- **الدمشقي** — ابن عامر
- **المدني الأول** — أبو جعفر

وهذا يعني أنّ **جميع رواة القراءة الواحدة يشتركون في ترقيم الآيات نفسه**، وكذلك القراءات التي تتبع نظام العدّ نفسه.

## المشكلة

لاختلاف أنظمة العدّ، لا تتطابق أرقام الآيات بين القراءات. فمثلاً: يعدّ حفص (الكوفي) (الم) في سورة البقرة آية مستقلة، بينما يدمجها ورش (المدني الأخير) مع الآية التي تليها. وهذا يعني أنّ **الآية رقم 5 عند حفص هي الآية رقم 4 عند ورش** في السورة نفسها.

لذلك فإنّ مطابقة الآيات بين المصاحف عن طريق رقم السورة ورقم الآية فقط تُعطي نتائج **خاطئة** في كثير من الأحيان.

## الحل

يوفِّر هذا المشروع **ملفات JSON جاهزة للاستخدام** تربط أرقام الآيات بين أنظمة العدّ المختلفة. لا تحتاج إلى كتابة أي كود، فقط حمّل ملفات JSON في أي لغة برمجة.

## أبرز الفروقات

تتمحور الفروقات بين أنظمة العدّ حول:

1. **البسملة** — يعدّها العدد الكوفي آية في سورة الفاتحة (الآية الأولى). أما بقية الأنظمة فلا تعدّها.

2. **الحروف المقطعة** — الحروف التي تبدأ بها 29 سورة (الم، الر، حم، إلخ). يعدّها العدد الكوفي آيات مستقلة، بينما تدمجها الأنظمة الأخرى مع ما يليها.

3. **التعويض بالتقسيم** — عندما يدمج نظام ما البسملة في الفاتحة، فإنه يقسم الآية السادسة (عند حفص) إلى آيتين للحفاظ على العدد المتفق عليه وهو 7 آيات.

## مثال سريع

### قراءة خريطة الربط

لمعرفة ما يقابل الآية 5 من سورة البقرة (عند حفص) في عدّ ورش (المدني الأخير):

```python
import json

with open('data/mappings/hafs-to-madani-last.json') as f:
    mapping = json.load(f)

entry = mapping['surahs']['2']['ayahs']['5']
print(entry)
# {"target_ayah": 4, "status": "mapped"}
```

### التحقق من الدمج

```python
entry = mapping['surahs']['2']['ayahs']['1']
print(entry)
# {"target_ayah": 1, "status": "merged"}
# (الم) مدمجة مع الآية التالية — target_ayah يشير إلى الآية المدمجة فيها
```

### التحقق من التقسيم

```python
entry = mapping['surahs']['1']['ayahs']['6']
print(entry)
# {"target_ayah": 5, "status": "split", "splits_into": [5, 6]}
# هذه الآية تنقسم إلى آيتين في النظام المستهدف
```
